pub struct Dense { /* fields omitted */ }

Defines a dense (or fully connected) layer.

This layer can be used as a conventional dense layer in feedforward neural networks or as a fully connected layer in convolutional neural networks. If the inputs of the layer are not one-dimensional, they are reshaped before being used to compute the activations.
