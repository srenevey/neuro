Initializers

Parameters initialization plays a very important role in deep learning. A proper initialization can determine wether the optimization algorithm will converge at all, and influences the convergence speed. Symmetry breaking is also an important part of the initialization process. If two units are connected to the same input and with the same initial weights and activation function, then their weights will be updated identically during the optimization process, assuming a deterministic loss function is used. This means that the units will learn the same function, thus reducing the overall ability of the network to learn complex functions. In order to avoid this symmetry, we need to break it by initializing the parameters randomly such that each unit has different weights and biases.
